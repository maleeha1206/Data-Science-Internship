# Data Science Internship Portfolio

Welcome to my master data science repository. This portfolio showcases distinct analytical pipelines built during my data science internship. The projects advance from fundamental exploratory visualization to supervised structural classification and continuous value regression.

---

## 📂 Portfolio Directory & Project Breakdown

### 📊 [Task 1: Iris Dataset Exploratory Analysis](./Task1/)
* **Objective:** Isolate structural geometric properties across botanical sub-species using visual statistics.
* **Core Skills:** Distribution profiling, pair-plotting, matrix correlations, outlier identification.
* **Tech Stack:** Matplotlib, Seaborn, Pandas.

### 💳 [Task 2: Credit Risk Classification](./Task2/)
* **Objective:** Build a predictive framework to determine applicant creditworthiness and minimize default frequencies.
* **Core Skills:** Multiclass probability mapping, data cleaning, train/test splitting.
* **Tech Stack:** Logistic Regression, Scikit-Learn.

### 📉 [Task 3: Bank Customer Churn Prediction](.Task3/)
* **Objective:** Map consumer attrition vectors and extract organizational operational vulnerabilities.
* **Core Skills:** Non-linear decision pathways, categorical one-hot encoding, feature importance ranking.
* **Tech Stack:** DecisionTreeClassifier, Scikit-Learn.

### 🩺 [Task 4: Medical Insurance Cost Projection](./Task4/)
* **Objective:** Estimate absolute continuous pricing charges driven by consumer demographic variables.
* **Core Skills:** Continuous line-of-best-fit optimizations, residual error diagnostics (MAE & RMSE).
* **Tech Stack:** LinearRegression, Scikit-Learn.

### 🔮 [Task 5: Capstone - Personal Loan Campaign Optimization](./Task5/)
* **Objective:** Forecast conversion probabilities for targeted bank marketing outreach.
* **Core Skills:** Micro-demographic customer segmentation, interactive prediction simulation engines.
* **Tech Stack:** DecisionTreeClassifier, Pandas, Seaborn.
